# 成语接龙 MVP 开发计划

## 目标

先完成一个单机学习型成语接龙：玩家输入成语，系统校验合法性并由 AI 自动接龙，同时展示拼音、释义、例句和本局学习总结。

## 第一版范围

- Vue 3、TypeScript、Pinia、Vite。
- 本地小型成语数据，后续可替换为完整 JSON 词库。
- 严格同字接龙。
- 已使用成语不可重复。
- AI 三档难度：轻松、普通、挑战。
- 提示候选、收藏生词、本局总结。

## 后续扩展

- 引入更完整的成语数据源。
- 增加词典搜索页和生词复习页。
- 增加同音接龙模式，并处理多音字。
- 增加本地统计，如提示次数、失败字、常错成语。
